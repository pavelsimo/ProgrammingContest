Problem Statement

    Krolik found a sheet of paper with an equation containing three positive integers: "a * b = c". This equation might be wrong, so Krolik is trying to make it correct by adjusting the numbers a, b, and c.



You are given three ints a, b, and c. Return a int containing the minimum value of |A - a| + |B - b| + |C - c|, where A, B, and C are positive integers satisfying A * B = C.

Definition

    
Class:CorrectMultiplicationTwo
Method:getMinimum
Parameters:int, int, int
Returns:int
Method signature:int getMinimum(int a, int b, int c)
(be sure your method is public)
    

Notes

-|x| is the absolute value of x (which is x for non-negative x and (-x) for negative x).

Constraints

-a, b, and c will each be between 1 and 1,000,000, inclusive.
